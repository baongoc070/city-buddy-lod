# CityBuddy
CityBuddy là người bạn đồng hành thông minh dựa trên nền tảng dữ liệu mở liên kết (LOD) mức độ 5 để kiến tạo môi trường sống bền vững. Tuân thủ các tiêu chuẩn quốc tế NGSI-LD, ontology SOSA/SSN và kế thừa các mô hình dữ liệu chuẩn từ FiWARE Smart Data Models.
